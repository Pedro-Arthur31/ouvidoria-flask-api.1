# Sistema de Ouvidoria API

API REST desenvolvida em Python utilizando Flask para gerenciamento de reclamações, respostas e usuários.

## Tecnologias

- Python
- Flask
- SQLAlchemy
- MySQL (Aiven)
- JWT Authentication
- Swagger
- Render
- Git/GitHub

## Funcionalidades

- Cadastro de usuários
- Login com JWT
- CRUD de usuários
- CRUD de reclamações
- CRUD de respostas
- Documentação Swagger
- Banco MySQL na nuvem
